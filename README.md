# Ракурс — сайт-визитка школы фотографии

Одностраничный сайт (плюс страница-галерея) для вымышленной онлайн-школы
фотографии «Ракурс». Чистый статический сайт: HTML/CSS/JS без сборки и
зависимостей.

## Страницы

- **`index.html`** — лендинг: шапка, первый экран, курсы, тарифы и цены,
  контакты. Есть избранные курсы (сохраняются в `localStorage`) и подсветка
  активного пункта меню при скролле.
- **`gallery.html`** — примеры кадров в виде **рамки-видоискателя**: HUD камеры,
  сетка по третям, рамка автофокуса. Слайдеры **Свет** и **Фокус** меняют кадр
  вживую (яркость/размытие), клик по рамке AF имитирует автофокус.

## Темы

Светлая и тёмная тема (переключатель в шапке). По умолчанию берётся системная
тема, выбор сохраняется в `localStorage`. Вся палитра — на CSS-переменных.

## Запуск

Открыть `index.html` в браузере или поднять статический сервер:

```bash
python -m http.server 5500   # затем http://localhost:5500
```

## Данные

Тексты, курсы и цены взяты из демо-документов базы знаний (`.docx`; в
репозиторий не входят); цены и условия вымышленные. Подробнее об архитектуре —
в [CLAUDE.md](CLAUDE.md).
